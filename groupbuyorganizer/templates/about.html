About Group Buy Organizer


Group Buy Organizer is an open-source web application developed by me, Mark Michon to streamline the group buying process for large wholesale orders. Spend less time coordinating and more time getting on with life.

This was developed entirely from production-grade libraries for maximum performance, stability, and security. The source code for this application can be viewed here.
